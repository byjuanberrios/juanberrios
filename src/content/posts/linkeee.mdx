Creé una aplicación sencilla para gestionar mis marcadores (*links que me interesan*). 
Bueno, decir creé es un poco exagerado, porque en realidad utilice la IA para crear la aplicación.

![Screenshot de la aplicación Linkeee](https://storage.juanberrios.com/linkeee_screenshot.webp "Screenshot de la aplicación Linkeee")

La aplicación <a href="https://github.com/byjuanberrios/linkeee" rel="noreferrer noopener" target="_blank">se puede encontrar en GitHub</a>. y la idea 
es que si te interesa, puedas alojarla gratuitamente en **Vercel** usando una base de datos en **Supabase** y autenticación con **GitHub.**

## El Porqué

Anteriormente, estaba usando [Linkding](/posts/linkding) para guardar mis marcadores. Pero tenía que pagar por un servidor y alojar unabase 
de datos.

Sin embargo para el uso mínimo que le di, no tenía sentido.

Así que necesitaba algo que me permitiera guardar links de manera esporádica y a la vez poder acceder a ellos a través de una API 
(*para poder incluir los bookmarks en este sitio web de manera estática*). Todo esto totalmente plausible con las tecnologías que ya te mencioné.

Así que con un par de comandos y el ajuste de pequeños detalles a través de *Cursor* se logró algo bastante decente.

Al ser una aplicación bien sencilla y personal solo me preocupé de que todo funciona correctamente. 
Ahora, supongo, queda el trabajo de optimizar el código, refactorizar ciertas cosas de manera manual 
o incluso con más iteraciones de un agente de IA.
